# vuepress-plugin-musicplayer

## Introduce

musicplayer plugin for vuepress theme.

是一款基于网易云API和`zw-player`播放器魔改来的。

## How to use

1. install `npm i vuepress-plugin-musicplayer -S`

2. add in `config.js`

```js
module.exports = {
  
  plugins: [
    ['vuepress-plugin-musicplayer', {
      isAllRank: true
    }],
  ]
}
```
-  `isAllRank` 展示所有排名歌单，默认`false`

## Tip

- This plugin supports `vuepress 1.x`
